Guia

Aqui você aprenderá sobre as Especificações das peças e como montar seu computador.

Componentes principais:

Placa-mãe

A placa-mãe é a peça central onde todos os outros componentes se conectam. Ela define quais processadores, memórias e outros dispositivos são compatíveis com o sistema. É importante verificar a compatibilidade de todos os componentes com a placa-mãe antes de comprá-los.

Lista de Placas-mãe

Processador (CPU)

O processador é o cérebro do computador. Ele realiza os cálculos necessários para a execução dos programas. A escolha do processador dependerá do uso pretendido. Processadores de alto desempenho são recomendados para jogos e edição de vídeo, enquanto modelos intermediários podem ser suficientes para tarefas cotidianas.

Lista de Processadores

Memória RAM

A memória RAM é o local onde o computador armazena dados temporários para acesso rápido. Quanto maior a capacidade de RAM, mais tarefas o computador poderá executar simultaneamente sem perder desempenho.

Lista de Memórias-RAM

Armazenamento (HD ou SSD)

O disco rígido (HD) ou a unidade de estado sólido (SSD) são os dispositivos responsáveis por armazenar o sistema operacional, programas e arquivos. Os SSDs são mais rápidos, mas os HDs costumam oferecer mais espaço a um preço menor. Muitas pessoas optam por combinar ambos: SSD para o sistema e programas, e HD para arquivos maiores.

Lista de SSDs e HD

Placa de vídeo (GPU)

A placa de vídeo é responsável por processar e gerar as imagens exibidas no monitor. É um componente crucial para quem utiliza o computador para jogos ou trabalhos gráficos intensos, como modelagem 3D e edição de vídeos.

Lista de Placas de vídeo

Fonte de alimentação (PSU)

A fonte de alimentação converte a energia da tomada em energia utilizável pelos componentes internos do computador. É importante escolher uma fonte de alimentação que forneça potência suficiente para todos os componentes, com alguma margem de segurança.

Lista de Fontes

Gabinete

O gabinete é a carcaça que abriga todos os componentes do computador. Além de estética, o gabinete deve fornecer uma boa ventilação para manter o sistema refrigerado e suportar as dimensões da placa-mãe e outros componentes.

Lista de Gabinetes

Coolers e sistema de refrigeração

Manter o computador em temperaturas adequadas é essencial para seu funcionamento estável. Além do cooler do processador, pode ser necessário adicionar ventoinhas extras ou até sistemas de refrigeração líquida para manter temperaturas seguras, especialmente em computadores de alto desempenho.

Lista de Coolers
